Twilio Notifier
===============

Provides Twilio integration for Symfony Notifier.

DSN example
-----------

```
// .env file
TWILIO_DSN=twilio://ACCOUNT_SID:AUTH_TOKEN@default?from=FROM
```

where:
 - `ACCOUNT_SID` is your Twilio account SID
 - `AUTH_TOKEN` is your Twilio auth token
 - `FROM` is the sender
